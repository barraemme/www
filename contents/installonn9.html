Install instructions

Snowshoe is available for Nokia N9 through its own repository, which also hosts Qt5 and QtWebKit packages. To install, follow these steps:

Step 1 - Enable developer mode

You can find how to do this on MeeGo 1.2 Harmattan Developer Documentation.

With the developer mode enabled you now have a terminal available to you under your N9 menu, open the terminal and install wget (required by the repository installer).

# apt-get install wget

Step 2 - Install Snowshoe GPG keys

Download and add the keys to avoid warnings when installing Snowshoe packages

# cd /tmp
# wget http://snowshoe.qtlabs.org.br/apt/conf/Qt5N9.gpg.key
# apt-key add Qt5N9.gpg.key
# rm Qt5N9.gpg.key

Step 3 - Install repository package and all dependencies

Snowshoe is built using a recent QtWebKit revision and the latest Qt5 hashes announced on QtWebKit mailing list, to install all dependencies first download and isntall the repositories package, this package will add Snowshoe repositories to your N9, then you can install Snowshoe.

# wget http://snowshoe.qtlabs.org.br/apt/snowshoe-repositories.deb
# apt-get install webkit-snapshot

Step 4 - Compile yourself or install a package

If you already have a scratchbox setup on your machine is very easy to compile and run snowshoe on your n9, just type (on your machine!):

$ cd <snowshoe dir>
$ ./tools/build-deploy.sh -i <device ip> -r

But if you want to just install the package and forget about scrachbox stuff, just install the snowshoe package.

# apt-get install snowshoe-mobile
