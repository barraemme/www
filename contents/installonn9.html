Install instructions

Snowshoe is available for Nokia N9 through its own repository, which also hosts Qt5 and QtWebKit packages. To install, follow these steps:

  • Enable developer mode on your N9. Within developer mode screen, install the "Utilities" set of packages, which contains wget, required by the repository installer.
  • Install the repository package. It'll add repository to the N9 software sources list.
  • With a package manager like MeeCatalog, install Snowshoe Mobile. The package will add the Qt5 and QtWebKit dependencies.

Snowshoe is built using a recent QtWebKit revision and the latest Qt5 hashes announced on QtWebKit mailing list.
